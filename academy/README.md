# MarketMind AI Academy

The academy is the narrative learning layer of MEKS. It turns canonical topics
into progressive, product-connected learning experiences.

## Rules

- Teach through MarketMind AI examples and production constraints.
- Link each chapter to canonical manifests in `knowledge/topics/`.
- Do not duplicate architecture decisions or implementation documentation.
- Support progression from beginner through architect.
- Include exercises, failure scenarios, interview questions, and evidence.
- Develop chapters incrementally while the corresponding product capability is
  built or operated.

## Existing Learning Areas

- Start here and skills map

- Java and Spring Boot

- Clean architecture, DDD, and hexagonal architecture
