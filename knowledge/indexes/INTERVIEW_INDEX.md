# Interview Index

## Question Graphs

- Fundamentals and implementation questions live with each topic.
- Cross-topic prompts live in `interview/`.
- Full design exercises link to the [System Design Index](./SYSTEM_DESIGN_INDEX.md).
- Behavioral evidence should reference real MarketMind decisions and outcomes.

## Assessment Levels

| Level | Interview signal |
|---|---|
| Beginner | Correct definitions and guided implementation |
| Intermediate | Independent implementation and debugging |
| Senior | Trade-offs, failure handling, and operational ownership |
| Staff | Cross-team design and reusable standards |
| Principal | Strategy, organizational leverage, and long-term risk |
| Architect | Business, product, data, platform, and governance synthesis |

Use `templates/INTERVIEW_TEMPLATE.md` for new question sets.
