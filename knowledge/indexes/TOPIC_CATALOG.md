# Topic Catalog

The machine-maintained catalog is declared in
`scripts/meks/generate_meks.py`. Canonical manifests are grouped under:

- engineering foundation;
- Java and runtime;
- Spring;
- APIs and identity;
- distributed systems;
- data;
- platform and cloud;
- delivery;
- observability and production;
- testing and security;
- AI;
- finance;
- leadership;
- interviews;
- product development;
- MarketMind modules.

Run the scaffolder after adding a topic. Developed files are preserved because
the generator creates missing files only.
